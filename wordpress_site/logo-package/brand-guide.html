Logo Usage Guidelines

Logo Variations

Horizontal Logo
Primary usage for headers

Vertical Logo
For square spaces

Wordmark Only
When icon isn't suitable

Icon Only
Favicons and small spaces

Clear Space & Scaling

Minimum Clear Space: Equal to the height of the shield icon

Minimum Size: 120px wide for horizontal, 60px wide for vertical

Maximum Size: No limit, but maintain readability

Usage Do's and Don'ts

✓ DO

  • Use on dark backgrounds
  • Maintain proper clear space
  • Use approved color combinations
  • Keep proportions intact
  • Use high-resolution files

✗ DON'T

  • Stretch or distort the logo
  • Change the colors
  • Use on light backgrounds without adjustment
  • Add effects or shadows
  • Use below minimum size

Brand Applications

Digital Applications

Website Headers

Use horizontal logo at 180-240px width on dark navigation bars.

Social Media

LinkedIn: Use provided banner (1584x396) and profile image (400x400)

Email Signatures

Horizontal logo at 150px width, linking to firstdatasecurity.com

Favicons

Print Applications

Business Cards

Vertical logo preferred, minimum 0.75" wide

Letterhead

Horizontal logo in top-left corner, 2" wide maximum

Presentations

Use horizontal logo in slide masters, consistent size throughout

Color Contrast Guidelines

WCAG AA Compliant Combinations:

  • Off-white (#F8FAFC) on Ink (#0E1116) - Contrast ratio: 15.8:1
  • Off-white (#F8FAFC) on Slate (#0F172A) - Contrast ratio: 14.2:1
  • Cyan (#26C6DA) on Ink (#0E1116) - Contrast ratio: 8.9:1
  • Emerald (#10B981) on Ink (#0E1116) - Contrast ratio: 7.2:1

File Naming Convention

Format: fds-logo-[variation]-[size].[extension]

Examples:

  • fds-logo-horizontal-1024.png
  • fds-logo-vertical.svg
  • fds-logo-icon-512.png
  • fds-logo-wordmark.svg
